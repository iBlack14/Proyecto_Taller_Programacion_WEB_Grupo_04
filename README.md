# Proyecto_Taller_Programacion_WEB_Grupo_04
Web creada en el curso de Taller de programacion WEB, por el grupo 04, UTP - Chiclayo 2026 - I

## Descripción

Sitio web institucional para **Game Center Chiclayo**, un centro de entretenimiento 
especializado en videojuegos ubicado en Alfredo Lapoint 853, Chiclayo. 
La plataforma busca proyectar la identidad visual de la marca (rojo y negro) 
y destacar su propuesta de valor: salas multijugador privadas, ambiente seguro 
y tecnología de última generación.

## Páginas

- Nosotros
- Salas
- Promociones
- Galería
- Campeonatos
- Contacto

## Tecnologías

- HTML5
- CSS3 puro (Flexbox y Grid)
- Google Fonts
- Diseño responsivo con media queries
